This is the source code for my public academic website. Feel free to clone it and use it as a template.

1. Edit `index.typ` and `personal_info.typ` to your own content
2. Run `typst compile --features html --format html index.typ index.html` to compile it into HTML, or use the GitHub Actions workflow.

You don't need to
- Learn a programming language
- Use a specific software to publish/edit
- Maintain your own backend server
